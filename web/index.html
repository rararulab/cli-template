Agent Backend

Built-in agent execution layer for AI-powered CLI workflows, ready to extend.

🚀

CI/CD Ready

GitHub Actions with lint, test, and release pipelines preconfigured and working out of the box.

⚙️

Config System

Layered configuration with environment overrides, file-based defaults, and runtime merging.

📦

Release Automation

Conventional commits, changelogs, and version bumps handled automatically by release-plz.

Get Started

$ cargo generate rararulab/cli-template
$ cd my-awesome-cli
$ cargo run -- --help
  1. Generate your project from the template
  2. Replace placeholder names in Cargo.toml and CLAUDE.md
  3. Start building your CLI commands
